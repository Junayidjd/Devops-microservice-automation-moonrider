Welcome to Spring Boot CRUD Example - v2.0

This version includes enhanced search functionality with query parameters and error handling.

Search Products

Search products by name, price, or quantity using query parameters.

GET /products/search?name=example&price=10.0&quantity=5

Example:

http://localhost:9191/products/search?name=example
http://localhost:9191/products/search?price=10.0
http://localhost:9191/products/search?quantity=5
http://localhost:9191/products/search?name=example&price=10.0&quantity=5

Error Handling

If no products are found, a 404 Not Found response is returned.

If an internal server error occurs, a 500 Internal Server Error response is returned.
